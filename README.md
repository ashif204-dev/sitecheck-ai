# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ashif204-dev/sitecheck-ai.git
```

Go to the project

```bash
cd sitecheck-ai
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```

Open your browser and visit

```text
http://localhost:3000
```
